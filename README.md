# Dungeon Crawler Carl — Character Compendium

A comprehensive character reference for the **Dungeon Crawler Carl** series (Books 1–7) by Matt Dinniman.

## Contents

The compendium covers **73 characters** organized into four tiers:

- **Main Characters** — Carl, Princess Donut, Mordecai, Katia
- **Major Recurring Characters** — Mongo, Zev, Odette, Elle, Imani, Prepotente, and more
- **Supporting Characters** — Frank Q, allies, antagonists, and faction leaders
- **Notable Minor Characters** — NPCs, one-off crawlers, and background figures

Each entry includes species/race, role, first appearance, book range, class evolution (where applicable), a detailed visual reference description, and a narrative summary.

## Spoiler Warning

This compendium contains **major spoilers for the entire series** through Book 7 (*This Inevitable Ruin*), including character fates, plot twists, and endgame events.
